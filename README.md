# Pokémon FireRed / LeafGreen Perfected

We’ve seen Kanto time and time again. In Red/Blue/Yellow, Gold/Silver/Crystal, FireRed/LeafGreen, HeartGold/SoulSilver, Let’s Go Pikachu & Eevee. Despite this, we have never seen a complete version of Kanto with all core mechanics in tact, all dungeons and locations explorable, and extended Gen 1 Pokémon lines available.

This project builds the definitive version of Kanto. Core mechanics are polished, modern features and quality of life standards are incorporated. This should be the most fleshed out version of Kanto possible while embracing the spirit of the original games.

If you would like to build off of this project, you are welcome to fork it or reuse any code, so long as you include appropriate credits.

### Guiding Principles
* Changes have strong canonical justification or precedent
* Capable of trading with Gen 3 games (RSE/FRLG) on real hardware
* Compatible with PKHax
* As mechanics are changed, NPC dialog is updated
* All code is made for this project or comes from open-source projects and is appropriately credited below
* This project has an [open source license](/LICENSE.md) and may be freely used provided derivative works are also open source

### Features

National Pokedex
* [ ] Pokedex is reorganized so related Pokemon are grouped together
* [ ] Pokedex lists stats, learnsets, and more
* [x] All Evolution & Trade restrictions are removed #️⃣Deokishisu
* [x] All Evolution items are available during the main campaign #️⃣Deokishisu

<br>
Pokemon

* [ ] All Gen 1 & Gen 2 Pokemon can be caught and are on trainer teams
* [x] All Legendaries can be encountered & Event Tickets are available #️⃣Deokishisu
* [x] Legendaries respawn if defeated (rebattle the Pokemon League) #️⃣Deokishisu
* [ ] Surfing & Fishing is significantly more diverse across routes
* [ ] Wild Pokemon level curves are much better balanced
* [ ] Shiny odds are changed to 1/4096 and shiny charm is available

<br>
Moves

* [ ] Pokemon Learnsets are updated to resemble Gen 4
* [ ] Moves are updated to reflect how they work in later generations
* [x] TMs are Reusable #️⃣Deokishisu
* [ ] Many Move Tutors have been added

&nbsp;&nbsp;⚠️ Physical/Special split will not be implemented

<br>
Battles/Trainers

* [ ] Double Battles initiate when 2 trainers see you at once
* [x] Gym Leaders can be rebattled #️⃣Deokishisu
* [ ] Trainer Classes have signature Poke Balls custom sprites, and feature more items on their Pokemon

<br>
Items

* [x] Stat reducing berries are ported from Emerald #️⃣Deokishisu
* [ ] Gen 3 Poke Balls are much more plentiful
* [ ] Many more items respawn (like berry spots, items on beaches)
* [ ] Many consumable items are collected in increments of 3 or 5

<br>
Mechanics

* [x] Overworld Ability effects have been ported from Emerald #️⃣Deokishisu
* [ ] Breeding is possible with only 2 badges
* [ ] Breeding system has been overhauled with later generation changes
* [ ] Player can decorate their room with dolls (ported from Emerald)
* [ ] Following Pokemon exist
* [ ] Player can Fly to any route
* [ ] PC System has additional features
* [ ] Rock Smash can provide items including Fossils
* [ ] In many places menus and dialog are shortened
* [ ] GameBoy sounds plays the classic Kanto music

<br>
World-Building

* [ ] Tons of NPCs have updated dialog that better integrate into the Pokemon universe
* [ ] New NPCs have been added that sell items or help the trainer
* [x] Master Trainers make a return from the Let's Go games #️⃣Deokishisu
* [ ] Ribbons are incorporated from RSE games

<br>
Visuals & Audio

* [x] Pokemon Summary screen shows accurate met locations even outside Hoenn #️⃣Deokishisu
* [x] Hidden information about moves (Hidden Power, Nature Power, Weather Ball, Return/Frustration) is visible in the interface #️⃣Deokishisu
* [ ] Sprite animations and custom trainer sprites have been added

<br>
Options

* [x] `INSTANT` text speed has been added (defaults to `FAST`) #️⃣Deokishisu
* [x] Difficulty Modes are added to make the game easier or harder #️⃣Deokishisu
* [x] A Special Nuzlock Mode is added #️⃣Deokishisu

<br>

And much much more!

See [the complete list of planned features](/docs/FEATURES.md).
